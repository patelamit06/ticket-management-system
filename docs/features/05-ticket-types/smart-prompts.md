# Smart prompts – Ticket types

**Backend:** NestJS TicketTypes per event. Entity: eventId, name, price, quantity, maxPerOrder, availabilityStart/End. CRUD for organizer; public list for event. Validate capacity on order. DTOs and Swagger.

**Frontend:** Organizer manage ticket types per event; event detail page shows types and quantity selector (TicketTypeSelector). See docs/features/05-ticket-types/ for task lists.
