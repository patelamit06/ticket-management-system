# Smart prompts – Payments (Stripe)

**Backend:** NestJS Stripe integration. Create PaymentIntent for order; webhook to mark order paid and create tickets. Idempotent webhook. Optional platform fee. DTOs and Swagger.

**Frontend:** Checkout with Stripe Elements/Checkout; create PaymentIntent via backend, confirm with Stripe SDK; success/failure redirects. See docs/features/07-payments-stripe/ for task lists.
