# Feature folder template

Copy this folder (without the leading `_`) when adding a new feature. Each feature folder should contain:

- **frontend-tasks.md** – Checklist of frontend tasks with status (Not started / In progress / Done)
- **backend-tasks.md** – Checklist of backend tasks with status
- **smart-prompts.md** – Ready-to-use prompts for development (context, stack, acceptance criteria)

Update the parent [FEATURE_STATUS.md](../FEATURE_STATUS.md) when feature or task status changes.
