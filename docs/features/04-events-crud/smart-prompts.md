# Smart prompts – Events CRUD

**Backend:** NestJS Events module. Entity: name, description, location, startDate, endDate, bannerUrl, status, organizerId. CRUD for organizer; public GET (published only). Resource guard for owner. DTOs and Swagger.

**Frontend:** Public events list and detail; Organizer create/edit event form (banner upload), publish/unpublish, my events list. EventCard component. See docs/features/04-events-crud/ for task lists.
